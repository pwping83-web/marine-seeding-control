# 05. 사양 표 요약 (기기·웹·서버)

빠르게 인쇄·첨부하기 위한 **한 페이지 요약**입니다.

## 현장 단말(권장 클래스)

| 항목 | 권장 범위 |
|------|-----------|
| MCU | **ESP32** (WROOM/WROVER/S3 등) — **RAM ≥ 320KB** 급 이상 |
| 플래시 | 프로그램+TLS 여유 **≥ 4MB** 모듈이 무난 |
| GPS | NMEA 출력, **3.3V**, 활성 안테나 권장 |
| LTE | **한국 밴드·유심** 확인된 모듈(SIM7600 등) + **별도 전원 여유** |
| 전원 | 송신 순간 **1A 급 스파이크** 감안 DC-DC |
| 방수 | 현장 박스 **IP65+**, 글랜드 케이블 |

## 네트워크·운영

| 항목 | 내용 |
|------|------|
| 단말→서버 | **HTTPS** POST, 헤더 `X-Device-Ingest-Key` |
| 궤적 주기(권장) | **30초 ~ 2분** (요금·배터리·부하에 따라) |
| 살포 기록 | **이벤트 시** 전송 (버튼/센서) |
| 해상 LTE | **고정 반경 없음** — 연안 위주, 원양은 별도 통신 검토 |

## 관제 웹(이 프로젝트)

| 항목 | 기본·비고 |
|------|------------|
| 궤적 표시 | 토글 **「해상 기기(LTE) 궤적」** — DB 폴링 기본 **~12초** (`VITE_VESSEL_LTE_POLL_MS`) |
| 살포 마커 갱신 | Supabase 시 **자동** — `seed_drop_records` 폴링 기본 **~12초** (`VITE_SEED_DROP_POLL_MS`, 4초~120초) |
| 궤적 색 | **주황 실선** |
| 살포 점 | **색동그라미 마커** (`seed_drop_records`) |
| 신선도 안내 | LTE 최종 점 **25분** 초과 시 경고 문구 |

## 서버(Edge)

| API | 용도 |
|-----|------|
| `vessel-track-ingest` | 궤적 `points[]` |
| `telemetry-ingest` | 살포 `records[]` |

Secret 이름: **`DEVICE_INGEST_SECRET`** (Supabase Secrets)

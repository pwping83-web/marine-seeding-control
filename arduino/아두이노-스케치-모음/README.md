# 아두이노 스케치·문서 모음

`arduino/` 아래에서 **스케치(.ino)** 와 **단말용 마크다운 문서**를 한글 제목 폴더로 묶어 두었습니다.

## 왜 한글 폴더가 바깥에 있나요?

- **Arduino IDE**는 스케치북 경로에 **비ASCII(한글) 폴더**가 끼면 환경에 따라 빌드·경로가 깨질 수 있습니다.
- 그래서 **실제로 여는 폴더**는 맨 안쪽 **영문 이름**(`esp32-marine-telemetry` 등)입니다.
- 바깥 `아두이노-스케치-모음`·**한글 칸 이름**은 “책장 정리”용입니다.

## 나중에 한글 없이만 쓰고 싶을 때

1. 아래 표에서 쓸 스케치의 **가장 안쪽 폴더**만 통째로 복사합니다.  
   예: `esp32-marine-telemetry` 폴더만 → 다른 PC의 `문서/Arduino/` 등에 넣고 IDE에서 엽니다.
2. `문서/` 안의 `.md`는 그대로 두고 읽거나, 필요한 파일만 같이 복사합니다.

## 스케치 칸 (한글 제목 → 영문 스케치 폴더)

| 한글 칸 | 내용 | IDE에서 열 폴더 |
|---------|------|------------------|
| **해상-궤적-살포-통합** | 궤적 주기 + 살포 1건, 한 스케치 | `해상-궤적-살포-통합/esp32-marine-telemetry/` |
| **해상-궤적-살포-모터릴레이** | 위 기능 + 모터·릴레이·비상정지 | `해상-궤적-살포-모터릴레이/esp32-marine-telemetry-motor-relay/` |
| **함정-AI-안전-보조** | 명령 폴링·사이렌·SOS 등 (선택) | `함정-AI-안전-보조/esp32-marine-ai-safety/` |

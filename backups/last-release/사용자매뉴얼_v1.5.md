## 1. 시스템 개요

**해양 종자 살포 관제 시스템**은 남해 연안 해역에서 수행하는 해조류(해양 종자) 살포 작업을 실시간으로 모니터링하고, 작업 계획 및 기상 조건을 통합 관리하기 위한 웹 기반 관제 플랫폼입니다.

### 주요 목적

- 살포 선박의 실시간 위치 및 항적 추적
- 살포 이력(시각·좌표·상태) 기록 및 조회
- 7일 기상 예보 기반 작업 가능 여부 판단
- 월별 작업 일정 예약 및 현황 관리
- 선박에 대한 지령 신호 송신

### 시스템 구성 요약

```
[ 관제 담당자 PC ] ──── 웹 브라우저 ──── [ 관제 시스템 화면 ]
                                              │
                                   ┌──────────┴──────────┐
                                   │  실시간 관제 화면    │
                                   │  작업 계획 화면      │
                                   └─────────────────────┘
```

> **※ 시연 버전 안내**: 본 버전은 납품용 UI 프로토타입으로, 선박 위치·살포 데이터는 브라우저 내부 시뮬레이션 값입니다.  
> 실서비스 전환 시 실제 선박 GPS·DB 서버와 연동됩니다.

## 2. 시스템 접속 방법

| 항목 | 내용 |
|------|------|
| **접속 주소** | `http://127.0.0.1:5199` (내부망 배포 후 변경 예정) |
| **지원 브라우저** | Chrome 120 이상, Edge 120 이상 권장 |
| **최적 해상도** | 1280 × 800 이상 |
| **네트워크** | 내부망 전용 (인터넷 불필요) |

**접속 순서:**
1. 컴퓨터를 켜고 Chrome 또는 Edge 브라우저를 실행합니다.
2. 주소창에 접속 주소를 입력하고 Enter를 누릅니다.
3. 로그인 화면이 표시되면 정상 접속된 것입니다.

## 3. 로그인 화면

로그인 화면은 시스템 최초 진입 시 표시되며, 관리자 인증 후 관제 대시보드로 이동합니다.

### 화면 구성 요소

| 구성 요소 | 설명 |
|-----------|------|
| **이메일 입력** | 기관 내 계정 이메일 주소 입력 (예: `admin@agency.go.kr`) |
| **비밀번호 입력** | 담당자 비밀번호 입력 |
| **관제 시스템 접속** 버튼 | 클릭 시 인증 후 대시보드로 이동 |
| **시스템 상태** | 화면 하단에 위성 연결·서버 정상·암호화 활성 상태 표시 |

### 로그인 절차

1. 이메일 입력란을 클릭하고 계정 이메일을 입력합니다.
2. 비밀번호 입력란을 클릭하고 비밀번호를 입력합니다.
3. **「관제 시스템 접속」** 버튼을 클릭합니다.
4. "접속 인증 중…" 메시지가 잠시 표시된 후 대시보드로 자동 이동합니다.

> **시연 모드**: 이메일·비밀번호를 입력하지 않아도 버튼만 클릭하면 대시보드로 이동합니다.

## 4. 실시간 관제 화면

### 4-1. 화면 전체 구성

실시간 관제 화면은 크게 **좌측 사이드바**와 **우측 지도 영역**으로 구성됩니다.

```
┌─────────────────────┬────────────────────────────────────────┐
│  로고 · 시스템명    │  상단 헤더바 (실시간 관제 · 기상 정보)  │
├─────────────────────┼────────────────────────────────────────┤
│  [실시간 관제]      │                                        │
│  [작업 계획]  탭    │                                        │
├─────────────────────┤                                        │
│  통계 (살포/누적/속도)│         지 도 영 역                  │
├─────────────────────┤   (선박 아이콘 · 항적 · 살포 점)       │
│  기상 현황          │                                        │
├─────────────────────┤                                        │
│  선박 위치          │                                        │
├─────────────────────┤                                        │
│  선박 신호 송신     │                                        │
├─────────────────────┤                                        │
│  종자 살포 이력     │                                        │
│  (스크롤 목록)      │                                        │
├─────────────────────┤                                        │
│  [기간 필터] [CSV]  │                                        │
└─────────────────────┴────────────────────────────────────────┘
```

### 4-2. 사이드바 — 기상 현황

좌측 사이드바 상단에 현재 해역 기상 정보를 표시합니다.

| 표시 항목 | 설명 | 단위 |
|-----------|------|------|
| **풍속** | 현재 바람 속도 | knot(kt) |
| **풍향** | 바람이 불어오는 방향 (예: 남서, 북동) | — |
| **돌풍** | 순간 최대 풍속 | knot(kt) |
| **파고** | 파도 높이 | 미터(m) |
| **시정** | 가시 거리 | 킬로미터(km) |
| **기온** | 현재 기온 | °C |

- 왼쪽 **나침반 아이콘**의 파란 화살표가 현재 풍향을 가리킵니다.
- 풍속이 **16 kt 초과** 시 우측 상단에 주황색 **「강풍 주의」** 경고가 표시됩니다.

### 4-3. 사이드바 — 선박 위치

현재 작업 선박(제3해양살포함)의 실시간 위치 정보를 표시합니다.

| 표시 항목 | 설명 |
|-----------|------|
| **북위** | 선박의 위도 좌표 (소수점 4자리) |
| **동경** | 선박의 경도 좌표 (소수점 4자리) |
| **방위** | 선박 진행 방향 (0°~360°) |
| **실시간 / 연결 확인** | 위성 신호 수신 상태 표시 |

### 4-4. 사이드바 — 선박 신호 송신

관제 센터에서 작업 선박으로 지령 신호를 보내는 기능입니다.

| 버튼 | 기능 |
|------|------|
| **귀항 명령** | 선박에 항구 귀환 명령 송신 |
| **살포 시작** | 종자 살포 작업 개시 명령 송신 |
| **살포 중지** | 종자 살포 작업 정지 명령 송신 |
| **위치 보고** | 선박에 현재 위치 보고 요청 송신 |

**사용 방법:**
1. 해당 명령 버튼을 클릭합니다.
2. "전송 중…" 표시가 나타납니다.
3. 약 2~3초 후 **「수신확인」** 메시지로 바뀌면 선박이 신호를 수신한 것입니다.
4. 신호 전송 중에는 다른 명령을 보낼 수 없습니다.

### 4-5. 사이드바 — 종자 살포 이력

살포 작업이 수행된 이력을 시간 순으로 목록으로 보여줍니다.

| 열 | 설명 |
|----|------|
| **번호** | 구역 라벨 + 일련번호 (예: A01, B03, C12) |
| **시각** | 살포가 기록된 시각 (시:분:초) |
| **위도** | 살포 지점의 위도 |
| **경도** | 살포 지점의 경도 |

- 목록 왼쪽 **색상 띠**는 살포 시점 경과에 따른 구분색입니다 (아래 7장 색상 범례 참조).
- 가장 최근 살포 건은 **흰색 테두리** 강조 표시됩니다.
- **조회 기간 필터**: 하단 날짜 입력란으로 특정 기간만 조회할 수 있습니다.

#### 살포 구역 번호 부여 체계

살포 지점은 지도상 동서 위치에 따라 **A·B·C 구역**으로 자동 분류되며, 각 구역 내에서 순번이 매겨집니다.

| 구역 | 해역 위치 | 번호 예시 |
|------|-----------|-----------|
| **A 구역** | 서부 연안 (좌측) | A01, A02, A03 … |
| **B 구역** | 중부 연안 (중앙) | B01, B02, B03 … |
| **C 구역** | 동부 연안 (우측) | C01, C02, C03 … |

```
[ 지도 구역 구분 ]

A 구역          B 구역          C 구역
(서부)  │   (중부)         │   (동부)
A01 ●   │   B01 ●  B02 ●  │   C01 ●
A02 ●   │   B03 ●          │   C02 ●
        │                  │
────────┼──────────────────┼──────────
  서쪽                           동쪽
```

- 각 살포 점에는 지도 위에 **구역 라벨 배지(예: B03)** 가 직접 표시됩니다.
- 선박이 구역을 이동하면 자동으로 해당 구역 번호가 증가합니다.
- 검수 불일치 건도 동일한 번호 체계를 따르며, 지도상 검정 점으로 표시됩니다.

**CSV 저장:**
1. 날짜 필터를 설정합니다 (선택 사항).
2. **「CSV」** 버튼을 클릭합니다.
3. 현재 조회 기간의 이력이 엑셀(Excel) 호환 파일로 자동 다운로드됩니다.
   - 파일명 형식: `종자살포이력_YYYYMMDD.csv`
   - 인코딩: UTF-8 (한글 깨짐 없음)

### 4-6. 지도 영역 — 살포 점 라벨 및 색상 안내

지도 위 각 살포 점에는 **구역 라벨(A01, B03 등)** 이 작은 배지로 직접 표시됩니다.  
지도 우상단 **「살포 색상 안내」** 버튼을 클릭하면 색상 범례 패널이 열립니다.

| 아이콘/색상 | 의미 |
|-------------|------|
| 🔶 주황 삼각형 | 작업 선박 현재 위치 |
| 청록 점선 | 선박 항적 (지나온 경로) |
| **A01** ● 진한 빨강 | 최근 살포 (45일 이내) — A/B/C 구역 번호 표시 |
| **B03** ● 주황 | 약 3개월 전 살포 |
| **C01** ● 분홍 | 약 1년 전 살포 |
| ● 연분홍 | 약 2년 전 살포 |
| ● 회색 | 2년 이상 전 살포 |
| ● 검정 | 검수 불일치 또는 위치 누락 |

> **라벨 읽는 법**: `A01` = A구역(서부) 첫 번째 살포, `B12` = B구역(중부) 열두 번째 살포

### 4-7. 지도 영역 — 조작 방법

| 조작 | 방법 |
|------|------|
| **지도 확대** | 우측 **＋** 버튼 클릭 또는 마우스 휠 위로 스크롤 |
| **지도 축소** | 우측 **－** 버튼 클릭 또는 마우스 휠 아래로 스크롤 |
| **중심 이동** | 우측 **나침반(↑)** 버튼 클릭 |
| **초기화** | 우측 **새로고침(↺)** 버튼 클릭 |

## 5. 작업 계획 화면

### 5-1. 화면 전환 방법

좌측 사이드바 상단의 **「작업 계획」** 탭을 클릭하면 작업 계획 화면으로 전환됩니다.  
다시 **「실시간 관제」** 탭을 클릭하면 지도 화면으로 돌아옵니다.

### 5-2. 요약 지표

화면 상단에 4개의 핵심 지표를 한눈에 표시합니다.

| 지표 | 설명 |
|------|------|
| **7일내 작업가능** | 향후 7일 중 작업 가능으로 예측되는 일수 |
| **이달 예약** | 이번 달 예약된 작업 건수 |
| **이달 목표 달성** | 완료된 살포 수량 / 목표 수량 비율 (%) |
| **기상 연기** | 기상 악화로 연기된 작업 건수 |

### 5-3. 7일 기상 예보

오늘부터 7일간 날짜별 기상 예보와 작업 가능 여부를 표시합니다.

**기본 표시 (날짜 카드):**

| 표시 요소 | 설명 |
|-----------|------|
| 요일·날짜 | 해당 일의 요일과 날짜 |
| ✅ / ⚠ / ❌ | 작업 가능 / 주의 / 불가 |
| 기온 | 해당 일 예상 기온 |

**상세 보기 (카드 클릭 시):**

원하는 날짜 카드를 클릭하면 해당 날의 상세 기상 정보가 아래에 펼쳐집니다.

| 항목 | 설명 |
|------|------|
| 풍속 | 예상 풍속 및 풍향 |
| 파고 | 예상 파도 높이 (돌풍 포함) |
| 시정 | 예상 가시거리 / 강수 여부 |
| 기온 | 예상 기온 / 수온 |

- 기준치 초과 항목은 **노란색**으로 강조 표시됩니다.
- 작업 불가 이유가 있으면 하단에 요약 표시됩니다.
- 다시 클릭하거나 **「닫기」** 를 누르면 접힙니다.

**작업 가능 판단 기준:**

| 조건 | 기준값 | 초과 시 |
|------|--------|---------|
| 풍속 | 15 kt 이하 | 주의 또는 불가 |
| 파고 | 1.5 m 이하 | 주의 또는 불가 |
| 시정 | 5 km 이상 | 주의 또는 불가 |
| 강수량 | 5 mm 미만 | 주의 또는 불가 |

> 모든 조건이 기준 이하일 때만 **「작업 가능」** 으로 판정됩니다.

### 5-4. 이달 작업 캘린더

이번 달 날짜별 작업 현황을 달력 형식으로 표시합니다.

| 색상 점 | 의미 |
|---------|------|
| 🟢 초록 점 | 작업 완료 |
| 🔵 파란 점 | 작업 예약됨 |
| 🟡 노란 점 | 기상 연기 |
| 🔴 빨간 점 | 기상 불가 예보 |
| 청록 테두리 | 오늘 날짜 |

### 5-5. 작업 일정 목록

예정된 작업과 완료된 작업을 목록으로 관리합니다.

**예정 작업 목록 보기:**
- 각 작업 행을 클릭하면 상세 정보(목표 개체수·선박·해역·비고)가 펼쳐집니다.
- 다시 클릭하면 접힙니다.

**완료 작업 보기:**
- 목록 하단 **「완료된 작업 (N건)」** 버튼을 클릭하면 완료 이력이 펼쳐집니다.
- 각 완료 항목을 클릭하면 목표 대비 실적 달성률 막대 그래프가 표시됩니다.

### 5-6. 작업 가능 기준 / 조석 정보

오른쪽 패널에서 현재 기상 조건의 기준 충족 여부와 오늘 조석 예보를 확인합니다.

| 항목 | 설명 |

| 작업 기준 패널 | 풍속·파고·시정 현재값이 기준치 충족 여부 (✅/❌) |
| 오늘 조석 | 만조·간조 시각 및 높이 |

## 8. 주요 용어 해설

| 용어 | 설명 |
|------|------|
| **종자 살포** | 해조류 복원을 위해 해양 생물 종자를 특정 해역에 투하하는 작업 |
| **살포 이력** | 작업 선박이 종자를 투하한 시각, 위도, 경도 기록 |
| **항적** | 선박이 지나온 경로. 지도에 점선으로 표시 |
| **소나 (Sonar)** | 지도 화면에서 선박 주변을 회전하며 표시되는 레이더 스윕 표시 |
| **파고** | 파도의 높이. 단위는 미터(m) |
| **시정** | 맑은 정도. 얼마나 멀리 볼 수 있는지를 km로 표현 |
| **돌풍** | 갑자기 강해지는 순간 풍속. knot(kt)로 표현 |
| **만조** | 하루 중 조수(바닷물)가 가장 높은 시각 |
| **간조** | 하루 중 조수(바닷물)가 가장 낮은 시각 |
| **knot (kt)** | 해상 속도 단위. 1 kt = 시속 1.852 km |
| **검수 불일치** | 살포 기록과 현장 검수 결과 사이에 위치 또는 수량 차이가 있는 건 |
| **CVS 파일** | 엑셀에서 열 수 있는 데이터 파일 형식. 살포 이력 내보내기에 사용 |
| **내부망** | 외부 인터넷과 분리된 기관 내부 전용 네트워크 |

## 9. 자주 묻는 질문 (FAQ)

**Q. 화면이 멈추거나 데이터가 갱신되지 않습니다.**  
A. 브라우저에서 **F5** 키를 눌러 새로고침하시기 바랍니다. 그래도 문제가 지속되면 담당 IT 부서에 문의하십시오.

**Q. 살포 이력이 보이지 않습니다.**  
A. 사이드바 하단의 날짜 필터를 확인하십시오. 날짜가 설정되어 있으면 해당 기간 외 이력은 표시되지 않습니다. 날짜 입력란을 비워두면 전체 이력이 표시됩니다.

**Q. CSV 파일을 다운로드했는데 엑셀에서 한글이 깨집니다.**  
A. 엑셀에서 파일을 열 때 **파일 → 가져오기 → 텍스트(CSV)** 로 열고, 인코딩을 **UTF-8**로 선택하십시오. 또는 메모장(Notepad)으로 열어 다른 이름으로 저장 시 UTF-8 인코딩을 선택하신 후 다시 엑셀로 여십시오.

**Q. 지도에 표시되는 위치는 실제 위성 지도인가요?**  
A. 현재 버전은 시연용 SVG 지도입니다. 향후 카카오맵 또는 국가공간정보 지도 API 연동이 예정되어 있습니다.

**Q. 선박 신호 버튼을 눌렀는데 「수신확인」이 뜨지 않습니다.**  
A. 신호 전송에는 통신 지연이 있을 수 있습니다 (최대 4~5초). 잠시 기다려 주십시오. 반복해서 나타나지 않으면 통신 상태를 확인하십시오.

**Q. 로그인 화면에서 비밀번호를 잊어버렸습니다.**  
A. 시스템 관리자에게 문의하여 비밀번호 초기화를 요청하십시오.

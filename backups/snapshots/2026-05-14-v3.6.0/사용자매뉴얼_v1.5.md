## 1. 시스템 개요

**해양 종자 살포 관제 시스템**은 남해 연안 해역에서 수행하는 해조류(해양 종자) 살포 작업을 실시간으로 모니터링하고, 작업 계획 및 기상 조건을 통합 관리하기 위한 웹 기반 관제 플랫폼입니다.

### 주요 목적

- 살포 선박의 실시간 위치 및 항적 추적
- 살포 이력(시각·좌표·상태) 기록 및 조회
- 7일 기상 예보(단기 합성 + 선택적 중기 보강) 기반 작업 가능 여부 판단
- 월별 작업 일정 예약 및 현황 관리
- 선박에 대한 지령 신호 송신

### 시스템 구성 요약

```
[ 관제 담당자 PC ] ──── 웹 브라우저 ──── [ 관제 시스템 화면 ]
                                              │
                                   ┌──────────┴──────────┐
                                   │  실시간 관제 화면    │
                                   │  작업 계획 화면      │
                                   └─────────────────────┘
```

> **※ 시연 버전 안내**: 본 버전은 납품용 UI 프로토타입으로, 선박 위치·살포 데이터는 브라우저 내부 시뮬레이션 값입니다.  
> 실서비스 전환 시 실제 선박 GPS·DB 서버와 연동됩니다.

## 2. 시스템 접속 방법

| 항목 | 내용 |
|------|------|
| **접속 주소(시연)** | ① 내부망·LAN: `http://192.168.45.214:5111/` ② 공개 배포: `https://marine-seeding-control-git-main-pwping83-webs-projects.vercel.app/` |
| **지원 브라우저** | Chrome 120 이상, Edge 120 이상 권장 |
| **최적 해상도** | 1280 × 800 이상 |

1. 컴퓨터를 켜고 Chrome 또는 Edge 브라우저를 실행합니다.

3. 로그인 화면이 표시되면 정상 접속된 것입니다.

## 3. 로그인 화면

로그인 화면은 시스템 최초 진입 시 표시되며, 관리자 인증 후 관제 대시보드로 이동합니다.

### 화면 구성 요소

| 구성 요소 | 설명 |
|-----------|------|
| **이메일 입력** | 기관 내 계정 이메일 주소 입력 (예: `admin@agency.go.kr`) |
| **비밀번호 입력** | 담당자 비밀번호 입력 |
| **관제 시스템 접속** 버튼 | 클릭 시 인증 후 대시보드로 이동 |
| **시스템 상태** | 화면 하단에 위성 연결·서버 정상·암호화 활성 상태 표시 |

### 로그인 절차

1. 이메일 입력란을 클릭하고 계정 이메일을 입력합니다.
2. 비밀번호 입력란을 클릭하고 비밀번호를 입력합니다.
3. **「관제 시스템 접속」** 버튼을 클릭합니다.
4. "접속 인증 중…" 메시지가 잠시 표시된 후 대시보드로 자동 이동합니다.

> **시연 모드**: 이메일·비밀번호를 입력하지 않아도 버튼만 클릭하면 대시보드로 이동합니다.

## 4. 실시간 관제 화면

### 4-1. 화면 전체 구성

실시간 관제 화면은 크게 **좌측 사이드바**와 **우측 지도 영역**으로 구성됩니다.

```
┌─────────────────────┬────────────────────────────────────────┐
│  로고 · 탭(지도·계획)│  상단 헤더바 (구역 기록 · 바람 등)     │
├─────────────────────┼────────────────────────────────────────┤
│  요약 3칸           │  (본문) AI 요약 자막 — 한 바 후 30초 휴 │
├─────────────────────┤  Leaflet 지도 · 시각                   │
│  AI 기상 안전       │  하단: 기상 예측 타임라인(반투명)      │
│  (판정·나침반·수치) │  우상: 시연·고도화 / 우중: 색상 안내   │
├─────────────────────┤                                        │
│  선박 위치          │                                        │
├─────────────────────┤                                        │
│  선박 신호 4버튼    │                                        │
├─────────────────────┤                                        │
│  항로 네비·속도     │  (항상 펼침, 접기 없음)                │
├─────────────────────┤                                        │
│  살포 이력(스크롤)  │                                        │
├─────────────────────┤                                        │
│  날짜·CSV·가져오기  │                                        │
└─────────────────────┴────────────────────────────────────────┘
```

### 4-2. 사이드바 — AI 기상 안전 허브

상단 카드에 **안전 / 주의 / 긴급** 판정, 풍속·파고·강수·기온 요약 아이콘, (설정 시) **AI기상 위험 상황 자동 요약 리포트**(Groq 등 LLM 연동)가 표시됩니다.  
이 등급과 지도 하단 타임라인의 「지금 위험」은 **긴급 임계**와 **단기예보 첫 시간대** 판정을 더 보수적으로 합친 값으로 맞춰, 서로 다른 문구가 나오지 않도록 되어 있습니다.  
기상 데이터는 기상청 API 연동 또는 시연용 시뮬레이션에 따라 달라질 수 있습니다.

### 4-3. 사이드바 — 선박 신호 · 요약

**선박 신호 송신(2×2, 아이콘+라벨)**

| 버튼 | 기능 |
|------|------|
| **귀항 명령** | 긴급 회항 지령(연동 시 단말·사이렌 시나리오와 연계) |
| **살포 시작** | 살포 시작 지령(시연). 누른 뒤 **「살포 중」** 표시가 잠깐 반짝이듯 강조될 수 있음 |
| **살포 중지** | 살포 중지 지령(시연) |
| **위치 보고** | 위치 보고 요청 |

Supabase가 설정된 경우 `ship_command_logs`에 기록되며, 미설정 시에도 UI 시연이 가능합니다.

**요약 3칸**(신호 블록 위): 금일 살포·누적 건수·속도(노트).

### 4-4. 사이드바 — 항로 네비 · 속도 제어 (항상 표시)

- **접기/펼치기 없음** — 항상 같은 영역에 표시됩니다.
- 출발은 현재 선박 위치, 목적지는 좌표 입력 또는 **지도 클릭 모드**로 지정.
- **최적 경로 계산**: 암초·보호구역 등을 장애물로 한 A* 시뮬레이션 경로, 지도에 노란 점선 표시.
- **목표 속도**: 슬라이더·PID 파라미터 요약, 선박에 속도 명령 전송(연동 시).

### 4-5. 사이드바 — 종자 살포 이력 (필터·스크롤)

| 열 | 설명 |
|----|------|
| **구역** | 구역 라벨 배지 (예: A01, B03) |
| **시각** | 살포 기록 시각 |
| **위도·경도** | 살포 지점 좌표(열 헤더와 데이터 정렬 일치) |

- **날짜 필터**로 조회 기간을 정하면, 그 범위 안의 이력이 **목록 전체**로 나오며 영역 안에서 **세로 스크롤**됩니다. (한 화면에 다 안 들어가도 됩니다.)
- 가장 최근 건은 **테두리 강조**로 표시될 수 있습니다.

#### 금일 항적·보고·구역 면적

| 요소 | 설명 |
|------|------|
| **보고** 버튼 | 금일 항적 요약, CSV 내려받기, **PDF 출력**(브라우저 인쇄 대화상자에서 「PDF로 저장」) |
| **구역 추정 면적** | 살포 점이 **3개 이상**일 때, 살포점 외곽(Convex Hull) 근사 **ha** 표시(참고치, 정밀 측량 아님) |
| **현장 상황보고** | 입력·반영 시 당분간 그 값이 예보 기반 판단에 반영될 수 있음 |

#### 살포 구역 번호 부여 체계

살포 지점은 지도상 동서 위치에 따라 **A·B·C 구역**으로 자동 분류되며, 각 구역 내에서 순번이 매겨집니다.

| 구역 | 해역 위치 | 번호 예시 |
|------|-----------|-----------|
| **A 구역** | 서부 연안 (좌측) | A01, A02, A03 … |
| **B 구역** | 중부 연안 (중앙) | B01, B02, B03 … |
| **C 구역** | 동부 연안 (우측) | C01, C02, C03 … |

```
[ 지도 구역 구분 ]

A 구역          B 구역          C 구역
(서부)  │   (중부)         │   (동부)
A01 ●   │   B01 ●  B02 ●  │   C01 ●
A02 ●   │   B03 ●          │   C02 ●
        │                  │
────────┼──────────────────┼──────────
  서쪽                           동쪽
```

- 각 살포 점에는 지도 위에 **구역 라벨 배지(예: B03)** 가 직접 표시됩니다.
- 선박이 구역을 이동하면 자동으로 해당 구역 번호가 증가합니다.
- 검수 불일치 건도 동일한 번호 체계를 따르며, 지도상 검정 점으로 표시됩니다.

**CSV 저장:**
1. 날짜 필터를 설정합니다 (선택 사항).
2. **「CSV」** 버튼을 클릭합니다.
3. 현재 조회 기간의 이력이 엑셀(Excel) 호환 파일로 자동 다운로드됩니다.
   - 파일명 형식: `종자살포이력_YYYYMMDD.csv`
   - 인코딩: UTF-8 (한글 깨짐 없음)

### 4-6. 지도 영역 — 살포 점 라벨 및 색상 안내

지도 위 각 살포 점에는 **구역 라벨(A01, B03 등)** 이 작은 배지로 직접 표시됩니다.  
지도 우상단 **「살포 색상 안내」** 버튼을 클릭하면 색상 범례 패널이 열립니다.

| 아이콘/색상 | 의미 |
|-------------|------|
| 🔶 주황 삼각형 | 작업 선박 현재 위치 |
| 청록 점선 | 선박 항적 (지나온 경로) |
| **A01** ● 진한 빨강 | 최근 살포 (45일 이내) — A/B/C 구역 번호 표시 |
| **B03** ● 주황 | 약 3개월 전 살포 |
| **C01** ● 분홍 | 약 1년 전 살포 |
| ● 연분홍 | 약 2년 전 살포 |
| ● 회색 | 2년 이상 전 살포 |
| ● 검정 | 검수 불일치 또는 위치 누락 |

> **라벨 읽는 법**: `A01` = A구역(서부) 첫 번째 살포, `B12` = B구역(중부) 열두 번째 살포

### 4-7. 지도 영역 — 조작 방법

| 조작 | 방법 |
|------|------|
| **지도 확대** | 우측 **＋** 버튼 클릭 또는 마우스 휠 **위로** 스크롤 — Leaflet과 같이 **현재 화면 기준 한 단계**(`zoomIn(1)`)씩 확대됩니다. |
| **지도 축소** | 우측 **－** 버튼 클릭 또는 마우스 휠 **아래로** 스크롤 — **한 단계**(`zoomOut(1)`)씩 축소됩니다. |
| **중심 이동** | 우측 **나침반(↑)** 버튼 클릭 — GPS/시뮬에 맞춰 다시 맞춤할 때 **수동 줌 조작 상태도 초기화**됩니다. |
| **초기화** | 우측 **새로고침(↺)** 버튼 클릭 — 경로·살포 등에 맞춘 **맞춤 줌**을 다시 잡으며, **수동 줌 조작 상태도 초기화**됩니다. |
| **현재 줌** | 우측 버튼 옆에 **「줌 N」**(정수 줌 레벨)이 표시될 수 있습니다. |

#### 4-7-1. 항로 길안내(경로 설정) 모달에서 지도 찍기

- 사이드바 **항로 길안내** 등으로 **「항로 길안내」** 모달을 연 뒤, 「**항로 꼭짓점 찍기**」또는「**살포 예정 찍기**」를 누르면 **큰 모달은 숨겨지고** 지도만 보입니다. 지도에서 점을 찍거나(또는 살포 예정점을 찍은 뒤), 하단의 **「항로 설정 끄기」/「살포 예정 설정 끄기」**를 누르면 **다시 모달**이 나타납니다.
- 금일 항적이 지도에 켜져 있고 항로 찍기가 활성일 때, **번호가 붙은 꼭짓점 마커**를 **드래그**하면 해당 좌표가 갱신됩니다. 팝업에서 위도·경도를 고쳐 **「좌표 적용」**할 수도 있습니다.

### 4-8. 지도 오버레이 — 자막 · 기상 예측 타임라인 · 시연 · 고도화

| 요소 | 위치 | 설명 |
|------|------|------|

## 5. 작업 계획 화면

### 5-0. AI 살포 계획 패널

**작업 계획** 탭 선택 시 화면 **왼쪽**에 표시됩니다.

| 구역 | 내용 |
|------|------|
| **일일 살포 한도** | 기상(풍·파·시정 등)에 따른 시뮬 한도와 진행 바 |
| **최적 해역 TOP5** | 수심·수온·해류 시뮬 점수 기반 추천 목록 |
| **A* 경로** | 선택 해역까지 장애물(암초·보호구역) 회피 경로 요약 |
| **PID** | 목표 속도 유지용 파라미터 제안 및 아두이노용 코드 복사 |

### 5-1. 화면 전환 방법

사이드바 상단 **달력 아이콘(작업 계획)** 탭을 누르면 작업 계획 화면으로 전환됩니다.  
**지도 아이콘(실시간 관제)** 을 누르면 지도 화면으로 돌아옵니다.

### 5-2. 요약 지표

화면 상단에 4개의 핵심 지표를 한눈에 표시합니다.

| 지표 | 설명 |
|------|------|
| **7일내 작업가능** | 향후 7일 중 작업 가능으로 예측되는 일수 |
| **이달 예약** | 이번 달 예약된 작업 건수 |
| **이달 목표 달성** | 완료된 살포 수량 / 목표 수량 비율 (%) |
| **기상 연기** | 기상 악화로 연기된 작업 건수 |

### 5-3. 7일 기상 예보

오늘부터 7일간 날짜별 기상 예보와 작업 가능 여부를 표시합니다.  
**앞쪽 날짜**는 단기예보에 가깝게 합성되고, 기상청 **중기예보**를 받을 수 있으면(동일 `VITE_KMA_SERVICE_KEY` 등) **며칠 뒤**부터 풍속·파고 등이 중기 값으로 덮어써질 수 있습니다. 키가 없으면 모의 중기로 보강되며, 화면에 **기상청 중기예보** 표시와 새로고침이 있습니다.

**기본 표시 (날짜 카드):**

| 표시 요소 | 설명 |
|-----------|------|
| 요일·날짜 | 해당 일의 요일과 날짜 |
| ✅ / ⚠ / ❌ | 작업 가능 / 주의 / 불가 |
| 기온 | 해당 일 예상 기온 |

**상세 보기 (카드 클릭 시):**

원하는 날짜 카드를 클릭하면 해당 날의 상세 기상 정보가 아래에 펼쳐집니다.

| 항목 | 설명 |
|------|------|
| 풍속 | 예상 풍속 및 풍향 |
| 파고 | 예상 파도 높이 (돌풍 포함) |
| 시정 | 예상 가시거리 / 강수 여부 |
| 기온 | 예상 기온 / 수온 |

- 기준치 초과 항목은 **노란색**으로 강조 표시됩니다.
- 작업 불가 이유가 있으면 하단에 요약 표시됩니다.
- 다시 클릭하거나 **「닫기」** 를 누르면 접힙니다.

**작업 가능 판단 기준:**

| 조건 | 기준값 | 초과 시 |
|------|--------|---------|
| 풍속 | 15 kt 이하 | 주의 또는 불가 |
| 파고 | 1.5 m 이하 | 주의 또는 불가 |
| 시정 | 5 km 이상 | 주의 또는 불가 |
| 강수량 | 5 mm 미만 | 주의 또는 불가 |

> 모든 조건이 기준 이하일 때만 **「작업 가능」** 으로 판정됩니다.

> **참고:** 실시간 관제 화면의 **시간대별 가능/주의/불가**(단기 슬롯 점수)는 `kma-weather.ts`의 **m/s·파고·강수확률** 등으로 따로 계산됩니다. 본 탭의 **15 kt** 기준과 숫자·등급이 다를 수 있습니다.

## 9. 자주 묻는 질문 (FAQ)

**Q. 화면이 멈추거나 데이터가 갱신되지 않습니다.**  
A. 브라우저에서 **F5** 키를 눌러 새로고침하시기 바랍니다. 그래도 문제가 지속되면 담당 IT 부서에 문의하십시오.

**Q. 살포 이력이 보이지 않습니다.**  
A. 사이드바 하단의 날짜 필터를 확인하십시오. 날짜가 설정되어 있으면 해당 기간 외 이력은 표시되지 않습니다. 날짜 입력란을 비워두면 전체 이력이 표시됩니다.

**Q. CSV 파일을 다운로드했는데 엑셀에서 한글이 깨집니다.**  
A. 엑셀에서 파일을 열 때 **파일 → 가져오기 → 텍스트(CSV)** 로 열고, 인코딩을 **UTF-8**로 선택하십시오. 또는 메모장(Notepad)으로 열어 다른 이름으로 저장 시 UTF-8 인코딩을 선택하신 후 다시 엑셀로 여십시오.

**Q. 지도에 표시되는 위치는 실제 위성 지도인가요?**  
A. **Leaflet 타일 지도**를 사용합니다. 타일 제공 URL·해상도는 배포 환경 설정에 따릅니다. 테스트 모드에서는 시뮬 선박·살포점이 함께 움직일 수 있습니다.

**Q. 선박 신호 버튼을 눌렀는데 「수신확인」이 뜨지 않습니다.**  
A. 신호 전송에는 통신 지연이 있을 수 있습니다 (최대 4~5초). 잠시 기다려 주십시오. 반복해서 나타나지 않으면 통신 상태를 확인하십시오.

**Q. 로그인 화면에서 비밀번호를 잊어버렸습니다.**  
A. 시스템 관리자에게 문의하여 비밀번호 초기화를 요청하십시오.

## 3. 화면 캡처 (PNG 또는 PDF)

- 생성 방법: 저장소 루트에서 `npm run capture:gov` → 기본 출력은 [`../captures/`](../captures/)  
- 제출하기 전: **스크린샷 속 개인정보·내부 URL** 마스킹 여부 확인  
- 한곳에 모아 두려면: 이 폴더 아래 [`캡처-또는-PDF-제출본-여기에-복사`](./캡처-또는-PDF-제출본-여기에-복사/) 에 **복사본만** 두고 첨부하면 혼동이 줄어듭니다.

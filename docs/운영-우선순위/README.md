### 📋 2-2. `.env` 실운영 키 설정

`docs/API-키-및-연동-가이드/README.md` 참고 후:

```env
VITE_KMA_SERVICE_KEY=   ← 기상청 단기예보 활용신청 후 발급
VITE_KMA_NX=            ← 실제 작업 해역 격자 X
VITE_KMA_NY=            ← 실제 작업 해역 격자 Y
DATA_GO_KR_SERVICE_KEY= ← 나라장터 입찰공고 (동일 포털에서 신청)
VITE_GROQ_API_KEY=      ← https://console.groq.com → API Keys (무료)
```

### 📋 2-3. 블루카본(잘피밭) R&D 재공고 모니터링

해양수산부 제2026-77호는 1차 접수 마감(2/19). 재공고 또는 2단계 공고를 대비:

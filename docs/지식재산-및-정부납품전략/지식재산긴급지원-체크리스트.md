# 지식재산 긴급지원·출원 연계 체크리스트 (일반)

신청서·계획서 PDF는 로컬에 보관하고, 여기서는 **빠짐 없이 준비할 항목 유형**만 정리한다.

## 신청·증빙 공통

- [ ] 신청서·사업계획서(해당 시) 최종본
- [ ] 중소기업·소상공인 등 **기업 확인** 서류
- [ ] 사업자등록 **증명**(정부24 등)
- [ ] 대표·담당자 **연락처 일치** 여부(신청 시스템·서류·이메일)
- [ ] **재무제표**(요구 기간·형식: 대차·손익 등 안내에 맞출 것)
- [ ] 출원·등록 관련 **통지·납부확인** (권리별 폴더 정리)

## 긴급지원 특유(사업 공고문 기준으로 매번 확인)

- [ ] 지원 **대상 권리**(특허·실용신안·상표·디자인·브랜드 등)와 신청 건수 제한
- [ ] **기업부담금·납부** 일정·증빙
- [ ] 연계 기관(지역 IP센터·테크노파크 등) **추가 서류** 여부

## 본 프로젝트(해양 관제)와 연결할 때

- [ ] 제품·서비스 설명이 **실제 납품 대상**(하드웨어·SaaS·유지보수)과 일치하는지
- [ ] 공공 제안서에 넣을 **권리 요약**(한 문단 + 권리표) 초안

구체 번호·날짜·금액은 `개인정보-로컬메모.md`에만 기록한다.

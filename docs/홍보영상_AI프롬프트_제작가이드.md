## 4. AI 도구별 사용 방법

### 🤖 Google Gemini / Veo (구글 AI 영상)

**접속:** [gemini.google.com](https://gemini.google.com) 또는 [labs.google/veo](https://labs.google/veo)

**입력 방법:**
1. 위 "한국어 버전" 프롬프트를 장면별로 복사·붙여넣기
2. "60초 홍보 영상을 만들고 싶어요. 다음 장면을 영상으로 만들어주세요:" 앞에 붙여서 입력
3. 생성 후 장면별 영상을 편집 프로그램에서 연결

**팁:** 프롬프트 끝에 `"4K, cinematic, professional, Korean government, no text overlay"` 를 추가하면 품질 향상

---

### 🎨 Fal.ai (팔.에이아이)

**접속:** [fal.ai](https://fal.ai) → `video-generation` 또는 `Kling` 모델 선택

**입력 예시 (장면 3):**
```
[Model: Kling 1.6 Pro 또는 Minimax Video]
Prompt: Close-up of a high-tech maritime control dashboard,
dark navy interface with glowing cyan data visualizations,
animated radar sweep, real-time vessel tracking,
Korean coastal map, professional control room,
cinematic 4K, teal glow

Duration: 5 seconds
Aspect Ratio: 16:9
```

**권장 모델:**
- `Kling Video 1.6 Pro` — 사실적 영상, 기술 장면에 적합
- `Minimax Video 01` — 긴 영상, 캐릭터 없는 장면에 적합
- `Stable Video Diffusion` — 자연 장면(바다, 해조류)에 적합

---

### 🎬 Runway Gen-4 (런웨이)

**접속:** [runwayml.com](https://runwayml.com) → `Gen-4 Video`

**사용 방법:**
1. 영문 프롬프트 입력
2. "Style Reference" 이미지에 `submission-dashboard.png` (현재 프로젝트 내 캡처본) 첨부
3. 대시보드 UI 스타일을 참조해 영상 생성

**팁:** Camera Motion 설정 → `Slow zoom in` (장면1,8) / `Static` (장면3) / `Aerial` (장면4)

---

### 🌐 Sora (OpenAI, 챗GPT 유료)

**접속:** [sora.com](https://sora.com) 또는 ChatGPT Plus → Sora 탭

**입력 예시:**
```
Create a 10-second cinematic video:

Large monitors display a dark navy dashboard with glowing cyan data points
representing seed drop locations on a coastal map.
A radar sweep rotates slowly. The room has a professional, authoritative atmosphere.
Color palette: deep navy, teal, electric blue.
Style: professional government documentary, 4K cinematic
```

# 초저가 티어 — 외부 전달용 복사본 전문 (단일본)

**한 파일에 모았습니다.** 아래 **「복사 구간」**만 순서대로 복사해 이메일·메신저·첨부 본문에 붙여넣으면 됩니다.  
**법률·특허 확정 문서 아님.** 제출 전 필요 시 변호사·변리사 검토.

## 사용 순서 (3단계)

1. **「1) 이메일 제목」** 블록 전체 복사 → 메일 제목란에 붙여넣기.  
2. **「2) 이메일 본문(안내)」** 블록 전체 복사 → 메일 본문 상단에 붙여넣기.  
3. **「3) 제안·사양 통합 본문」** 블록 전체 복사 → 같은 메일 본문 **이어서** 붙여넣기 **또는** 별도 첨부(txt/docx)로 저장.

**카카오톡·문자:** **「4) 초단축 요약」**만 보내고, 상세는 메일로 보낸다고 안내하면 됩니다.

## 3) 제안·사양 통합 본문 — 복사 시작

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
초저가 티어 — 과업범위·기술사양·검수 (통합)  v2.1  /  2026-05-15
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【0. 문서 성격】
본 문서는 예산·발주 설명을 위한 「최소 기능」안입니다.
풀스펙 관제·항로·대시보드·기계식 자동 파종 구동과는 별도이며,
협상 시 과업 범위를 한정하는 것을 목적으로 합니다.
본 문서 **단독으로는** 수신자와 제안자 간 **법적 구속력 있는 계약이 성립하지 않으며**,
구속력은 **쌍방 서명(또는 합의된 전자계약)된 계약서**에 따릅니다.

【0-보. 설치·운용 환경에 따른 가변(중요)】
구체 사양·통신 방식(WiFi/LTE)·안테나·전원·방수·검수 장소·일정·비용·납품 형태는
**해상(이동체 포함)·육상·기지·항만·실험실·창고 등 실제 설치·시험 환경**과
**전파·망 가용성·기상·안전 규정**에 따라 달라질 수 있습니다.
따라서 본 초안은 **참고용**이며, 최종 과업 내용은 **계약서·과업지시서·발주기관 요구사항 및
현장 실사·합의된 서면**에 의해 확정됩니다. 본 초안과 현장 조건이 다를 경우 **재견적·재협의** 대상입니다.

【1. 제안 한 줄 요약】
시연용 프로토타입 단말 1대로, 살포(투하) 시점의 위치 좌표만 서버에 기록하고,
엑셀(xlsx) 또는 CSV 파일로 변환·제공하는 것까지를 1차 과업으로 한정합니다.
구성은 상용 부품 조합·공개 예제 수준을 전제로 하며, 상용 해양 등급·연속 가동을 보증하지 않습니다.

【2. 포함 범위】
· 단말: ESP32 등 MCU(종류는 합의). 외장 GPS(UART). 살포 입력 1채널(GPIO 에지 또는 합의된 수동 트리거).
· 통신: 시연은 WiFi 우선. LTE 전용 경로는 별도 과업·별도 비용일 수 있음.
· 펌웨어: 살포 1건 발생 시 위도·경도·시각을 수집하여 서버로 전송(HTTP POST, 합의된 API).
  — 선박 주기 궤적(항적) 전송은 포함하지 않음.
  — 기계 모터·릴레이·비상정지 제어는 포함하지 않음.
· 서버: 점(좌표·시각 등)만 수신·저장 가능한 최소 API·저장소(시연·검수용).
· 납품 데이터: 엑셀 또는 CSV로 추출·전달, 간단 조작 설명 1~2쪽 수준.
· 시험: 실내 또는 합의된 장소에서 작동 시연·기본 통신 시험 1회(또는 합의 횟수).
· 납품 대수: 프로토타입 1대. 동일 사양 복제·양산은 별도 견적.

【2-보. TLE(궤도 요소) 관련】
문서상 「TLE 기본」이 필요하면 별도 마일스톤·유상으로 구분합니다.
현재 제공 예정인 통합 펌웨어 스케치에는 TLE 연산 코드가 없으며,
해당 기능은 추가 개발 과업입니다.

【3. 명시적 비포함】
· 관제 웹: 항로·네비게이션, 지도 위 실시간 항적, 멀티 사용자·권한, 통계·알림, 고급 UI.
· 하드웨어: 기계식 파종 구동 제어, 방수·내진 등 상용 해양 등급 설계, 대량 제작.
· 성능 보증: GPS 신호 불량·차폐·악천후·안테나 조건에 따른 연속 정확도 보증 없음.
· TLE: 미수신·오류 시 이중화·상용 유료 궤도 데이터 구독.
· 운영: 고가용성·장기 백업·대용량 트래픽·24시간 모니터링, 클라우드 월 비용 무제한 부담,
  SIM·도메인·SSL 비용 상한 없는 포함.
· 문서·교육: 상세 운영 매뉴얼, 다수 인력 교육, 장기 무상 유지보수.
· 현장 해상 시험·출장: 본 견가에 포함하지 않음(별도 비용, 발주 또는 수급사 주관).

【4. 비용·부담 주체(기본안)】
· 출장·숙박·교통·현장 인력: 발주 또는 수급사(계약상 갑).
· GPS 모듈·안테나·케이블·방수함체·SIM 등 추가 부품: 발주 또는 수급사, 또는 실비 정산(별도 합의).
· 본 범위 초과 기능·성능 개선(GPS·TLE 안정화, 양산, 웹 고도화): 추가 비용(별도 견적).

【5. 시스템 구성(개념)】
[GPS]—UART—[MCU]—WiFi 또는 LTE—[최소 수집 API]—[저장]—엑셀/CSV 추출
```

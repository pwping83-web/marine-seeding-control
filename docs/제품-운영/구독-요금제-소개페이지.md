## 현장 PLC 설치 — 기본 vs 특수

현장 **PLC·단말** 작업은 **월 구독과 별도**이며, 대부분 **기본설치**로 시작합니다. 선박 기계를 **특수하게** 묶어야 할 때만 **특수설치**로 넘어갑니다.

### 기본설치 (표준)

| 항목 | 내용 |
|------|------|
| **하드웨어** | **함체에 재작(개조)된 충전형 PLC** (ENX 표준 구성) |
| **목적** | GPS·살포·상태 신호를 **관제 웹 서버로 수신** (LTE·Telemetry·POST 등 표준 경로) |
| **포함 범위** | PLC·통신 설정, 관제 DB·대시보드와 **1척 연결** 완료까지 |
| **포함 안 됨** | 선박 **기계 자체**를 새 방식으로 돌리는 제어·타이머 배선 |
| **비용** | **₩500,000~800,000 / 척** (1회) + (필요 시) **출장비 별도** |

> 「관제 화면에 선박이 보이고, 살포·위치가 올라온다」까지가 **기본설치** 범위입니다.

### 특수설치 (협의)

| 항목 | 내용 |
|------|------|
| **대상** | **기계 부** — 릴·펌프·밸브·릴레이·**타이머** 등을 **특수 방식**으로 연동·구동하는 **PLC·코딩** |
| **예시** | 기존 장비 시퀀스에 맞춘 **비표준 타이밍**, 다단 릴레이, 현장만 있는 **구형 배선** 대응 |
| **비용** | **협의** — 회로·장비 사진·현장(또는 원격) 확인 후 **견적서** (보통 **₩100만 원~**, 착수 전 범위 확정) |
| **주의** | 기본설치만으로 되는지 **먼저 판단**합니다. 특수 요구를 기본 견적에 섞지 않습니다. |

```
[ 기본설치 ]  충전형 PLC ──수신──▶ 관제 웹 서버  (월 구독 SaaS)
[ 특수설치 ]  선박 기계부 ◀──PLC 제어──▶ (타이머·특수 로직)  → 별도 협의
```

## 자주 묻는 질문

**Q. 월 50만 원에 출장·설치도 되나요?**  
A. **아닙니다.** 월 구독은 **소프트웨어·클라우드 이용**입니다. 출장·연결·세팅은 **별도 견적**입니다.

**Q. 버그 고쳐 달라고 하면 돈이 드나요?**  
A. **제품·연동 결함**은 범위 내 **무상**으로 봅니다. **요구 변경·추가 화면·타 시스템 연동**은 **A/S·추가 코딩**으로 **유상** 견적합니다.

**Q. PLC 기본설치와 특수설치 차이는?**  
A. **기본**은 **충전형 PLC + 웹 서버 수신**까지입니다. **특수**는 **기계·타이머를 특수 방식으로 돌리는 PLC 코딩**이라 **범위·금액 협의**입니다.

**Q. 아직 단말이 없는데 월만 내면 되나요?**  
A. **설치 전**에도 **시연·시뮬**은 가능합니다. **실선 연동**은 **기본설치(또는 협의된 특수설치)·세팅** 후입니다.

**Q. 데모는 무료인가요?**  
A. **시연 URL**은 무료입니다. 기관 맞춤·현장·개발은 **계약 후**입니다.

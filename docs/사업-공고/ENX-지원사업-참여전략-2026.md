## 4. 블루카본 해초대 자동조성 R&D — 컨소시엄 참여 전략

| 항목 | 내용 |
|-----|------|
| **공고번호** | 해양수산부 공고 제2026-77호 |
| **사업명** | 블루카본(해초대)의 탄소중립 목표달성을 위한 대규모 자동 조성 기술개발 |
| **총 규모** | **134.5억 원** / 사업기간 4년 이내 |
| **1차 접수** | **이미 마감** (2026-02-19) |
| **주관** | 해양수산과학기술진흥원(KIMST) |
| **문의** | KIMST 생명환경팀 02-3460-0318 / khpark@kimst.re.kr |

**ENX의 포지셔닝 전략 (직접 신청 불가 → 컨소시엄 역할)**

이 과제는 **단일 지정공모·대형 컨소시엄** 구조입니다. 당사 단독 주관은 현실적으로 불가하지만, **선발된 주관기관(대학·출연연·KIOST 계열)의 공동·위탁 파트너**로 참여하는 경로가 있습니다.

당사 기술이 RFP와 겹치는 항목:
- **파종 모듈 위치 정확도(5m 이내)** → GPS Telemetry + 관제 웹 GIS
- **실해역 테스트베드 모니터링** → 실시간 대시보드
- **표준형 파종 데이터 수집** → 아두이노 GNSS 살포 연동

**다음 행동**
1. KIMST 생명환경팀에 **기술 소개 메일 + 시연 링크** 발송
2. 선발 가능성이 있는 기관(한국해양과학기술원 KIOST, 국립수산과학원, 수산자원공단)에 협력 제안
3. 재공모 여부 상시 모니터링 (KIMST 공지사항)

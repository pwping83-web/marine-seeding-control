## 1. “추후 코드 수정 가능한가?” — 요약

| 질문 | 답 (일반론) |
|------|-------------|
| 출원 후에도 코드를 바꿀 수 있나? | **가능하다.** 계수 튜닝·파고 반영 추가 등은 보통 계속한다. |
| 출원 명세와 다르게 바꿔도 되나? | **주의.** 출원 시 적은 내용과 **정반대** 동작이 되면 보정·진술 리스크가 있다. |
| 안전한 전략 | 출원 **직전**에는 **명세 = 그 시점 코드**에 맞춘다. 큰 동작 변경은 **문서 동시 갱신** 또는 변리사와 **보정 범위** 확인. |

## 2. 지금 출원할 때 (명세 문장 가이드)

- A*: **장애물 회피 격자 탐색** + 스텝 비용 **haversine(km) × 기상 배수(풍향·풍속·돌풍; `wx` 생략 시 1)**.  
- **구역 점수·일일 한도·PID**는 동일 모듈의 **별 함수**와 연동.  
- 복사용: [`발명의-구체적-실시예-전문.md`](./발명의-구체적-실시예-전문.md), [`출원-명세-합본-초안-복사용.md`](./출원-명세-합본-초안-복사용.md).

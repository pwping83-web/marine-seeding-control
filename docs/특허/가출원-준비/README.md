# 가출원(빠른 출원·우선일) 준비 자료

> **법률 자문 아님.** 한국 특허 제도에서 말하는 절차·용어(가출원·분할·우선권 등)는 **연도·출원 유형**에 따라 달라질 수 있으므로, **최종 확인은 [특허청(KIPO)](https://www.kipo.go.kr) 안내·특허로(온라인 출원) 도움말·변리사**로 한다.  
> 본 폴더는 **내부 보관·체크리스트·명세 초안 복사용**이다. 출원 자체를 대신하지 않는다.

## Git에 올리면 안 되는 것

- 출원번호, 접수일·시각을 제외한 **개인 식별·비밀**, 납부 영수증 스캔 원본 전체(필요 시 로컬만)  
→ **`가출원-로컬메모.md`는 `.gitignore`에 등록**되어 있다. 예시만 `*.example`으로 커밋한다.

# Cursor(또는 자문)에게 하나씩 물을 때

## 이렇게 물으면 답이 잘 나온다

1. **한 메시지에 질문 하나** — 예: “독립항 1을 시스템 청구로 쓸지 방법 청구로 쓸지”만.  
2. **사실 관계**를 한 줄 — 예: “지금 출원은 혼자 특허로에 넣을 예정이다.”  
3. **이미 정한 것**이 있으면 적기 — 예: “발명의 명칭은 OO로 고정했다.”

## 내가 답 못 하는 것 (변리사·특허청이 맞다)

- 이 출원이 **등록될지**, **거절이면 몇 %인지**  
- **가출원**이라는 말이 **지금 케이스에 법적으로 정확한지**(연도·출원 유형마다 다름)  
- **관할·수수료·기한**의 최종 숫자

## 레포에 두는 문서의 한계

- 여기 있는 글은 **초안·정리**이지, **출원서류 등본**이 아니다.  
- 출원 후에는 **특허로에 저장된 PDF**가 기준이 된다.

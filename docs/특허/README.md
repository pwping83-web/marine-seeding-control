## 변리사 제출용 ZIP 만들기 (문서만 전달)

1. 폴더 [`변리사-제출용/`](./변리사-제출용/)을 연다.  
2. 그 안의 **`docx/`** 에 Word 파일이 있다(아래 스크립트로 재생성 가능).  
3. **민감정보 금지:** API 키, 출원번호, 사업자번호, `.env` 내용은 넣지 않는다.  
4. 다음을 **한 ZIP**으로 묶어 메일·USB로 전달하면 된다.  
   - `01`·`02`·`03`의 **.md 또는 .docx** 중 편한 형식(변리사 선호 확인)  
   - `04` 안내에 적힌 **PNG 캡처·아키텍처 도면**(별도 파일로 첨부)

**DOCX 다시 만들기:** 저장소 루트에서  
`pip install python-docx` 후 `python scripts/patent-md-to-docx.py`  
또는 `npm run patent:docx`

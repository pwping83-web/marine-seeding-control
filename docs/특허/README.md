## 변리사 제출용 압축 파일 만들기 (문서만 전달)

1. [변리사 제출용](./변리사-제출용/) 폴더를 연다.  
2. 그 안의 **`워드 사본`(`docx/`)** 에 워드 파일이 있다(아래 스크립트로 다시 만들 수 있음).  
3. **민감정보 금지:** API 키, 사업자번호, 환경 설정 파일 내용은 넣지 않는다. (출원번호는 레포 [`가출원-접수-기록.md`](./가출원-준비/가출원-접수-기록.md)에 요약되어 있으나, 상대방 요청에 따라 전달본에서 생략할지 판단한다.)  
4. 다음을 **한 개의 압축 파일**로 묶어 메일·USB로 전달하면 된다.  
   - `01`·`02`·`03`의 **마크다운 또는 워드** 중 편한 형식(변리사 선호 확인)  
   - `04` 안내에 적힌 **PNG 캡처·아키텍처 도면**(별도 파일로 첨부)

**워드 문서 다시 만들기:** 저장소 루트에서  
`pip install python-docx` 후 `python scripts/patent-md-to-docx.py`  
또는 `npm run patent:docx`

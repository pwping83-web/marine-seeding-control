## 캡처 시 주의

- 브라우저 주소창에 **localhost 외 실제 배포 URL·쿼리 토큰**이 보이면 가리거나 다른 화면을 캡처한다.
- **지도 타일 API 키**가 URL에 노출되지 않게 한다.

## Mermaid 시퀀스 (선택)

변리사가 흐름을 물을 때를 대비해, 아래 내용을 한 장의 도면(PNG)으로 그려 넣을 수 있다.

- 단말 HTTPS POST → `telemetry-ingest` / `vessel-track-ingest`  
- 검증·레이트 리밋 → `seed_drop_records` / `vessel_track_points`  
- 관제 웹이 DB에서 조회·지도 반영

(텍스트만으로도 되면 본 레포 `관공서_시스템_아키텍처_구조도.md` 의 Mermaid를 참고한다.)

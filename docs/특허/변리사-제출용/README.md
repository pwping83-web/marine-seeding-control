# 변리사 제출용 패키지 (문서만 전달)

이 폴더만 압축(ZIP)하거나, 아래 파일들을 골라 이메일 첨부하면 **코드 저장소 전체를 주지 않고** 상담을 시작할 수 있도록 구성했다.

## 포함 파일

| 순서 | 파일 | 설명 |
|------|------|------|
| 01 | `01-미팅용-1페이지-발명-제안-요약.md` / `docx/01-….docx` | 1페이지 요약 + 질문 리스트 |
| 02 | `02-발명-제안서-초안-코드베이스정렬.md` / `docx/02-….docx` | 구현 근거·파일 경로가 있는 상세 초안 |
| 03 | `03-제출물-체크리스트.md` / `docx/03-….docx` | 빠짐없이 챙길 항목 |
| 04 | `04-도면-캡처-첨부-안내.md` | **별도 준비할 PNG·도면** 목록 (ZIP에 같이 넣기) |

## Word 파일

`docx/` 하위 `.docx`는 `python scripts/patent-md-to-docx.py` 로 Markdown에서 생성한다. 표·제목은 자동 변환이지만, 변리사가 원하면 **한글(HWP)**으로 옮겨 적어도 된다.

## 넣지 말 것

- `.env`, API 키, Supabase 서비스 롤 키, `DEVICE_INGEST_SECRET` 값  
- 출원번호·사업자등록번호·개인 주소·대표 전화 등 민감정보  
- **내부 운영 URL**이 풀린 스크린샷

## ZIP 만들 때 권장 구조

```
변리사-제출-YYYYMMDD.zip
├── 01-미팅용-1페이지-발명-제안-요약.docx
├── 02-발명-제안서-초안-코드베이스정렬.docx
├── 03-제출물-체크리스트.docx
├── figures/
│   ├── 시스템_아키텍처_구조도.png
│   ├── 항로_살포계획_화면.png
│   └── 관제_대시보드_예시.png
└── (선택) 04-도면-캡처-첨부-안내.pdf
```

`figures/` 안 PNG는 저장소에 없을 수 있으므로, 로컬에서 캡처·보내기 후 넣는다.
